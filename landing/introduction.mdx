## What is Particle Network?

Particle Network streamlines both user and developer experiences by abstracting away blockchain complexity. Its Wallet Abstraction SDKs offer one-click onboarding into smart accounts via social login, while the Universal Accounts SDK brings native chain abstraction, enabling a single user identity and unified balance across multiple chainsâ€”all through one account layer.
## Key Products

Particle Network current product suite features numerous SDKs made to simplify user interactions with dApps:

<CardGroup cols={2}>
  <Card title="Universal Accounts" icon="link" href="../ua-api-reference/ua-sdk/introduction">
    Chain Abstraction at the account level. Give users a unified identity and unified balance across multiple chains. 
  </Card>
  <Card title="Particle Connect" icon="link" href="../api-reference/connect/introduction">
    Unified modal aggregating both social logins (through Particle Auth) and
    wallet connection.
  </Card>
  <Card title="Particle Auth" icon="shield" href="../api-reference/auth/introduction">
    Onboard users with social authentication using popular platforms like
    Google, X, and so on.
  </Card>
  <Card title="Particle Wallet" icon="wallet" href="../api-reference/wallet/introduction">
    An application-embedded wallet modal used as a clean, predictable interface for displaying assets and driving basic account interaction.
  </Card>
  <Card
    title="Account Abstraction"
    icon="credit-card"
    href="../api-reference/aa/introduction"
  >
    Full-stack account abstraction SDK driving the implementation of smart
    accounts, including features like gasless transactions and session keys.
  </Card>
  <Card title="BTC Connect" icon="btc" href="../api-reference/btc/introduction">
    Connect Bitcoin-native wallets, such as UniSat or Xverse, to applications
    built on EVM-compatible chains.
  </Card>
